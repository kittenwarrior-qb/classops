# Quan sát buổi học

Mỗi buổi quan sát → một file `YYYY-MM-DD-lop.md`, dùng mẫu ở
`mau-quan-sat.md`.

**Các file quan sát nằm trong `.gitignore`** vì có thể chứa thông tin nhận
dạng người thật. Chỉ `README.md` và `mau-quan-sat.md` được commit.

## Cần bao nhiêu buổi

Tối thiểu 2. Tốt nhất 4, để thấy được cái lặp lại.

## Nguyên tắc

- Quan sát mạnh hơn phỏng vấn. Người ta không mô tả chính xác được việc
  mình làm hằng ngày
- Ghi lại cái nhìn thấy, không ghi cái mình suy ra
- Tách riêng phần "quan sát" và phần "diễn giải" khi viết
